# 🍔 Site para Restaurantes - Landing Page

Site completo e responsivo focado em captar clientes restaurantes (pizzarias, hamburguerias, cafés) que precisam de presença digital e cardápio online.

## 🎯 Objetivo

Landing page de conversão para um estúdio web especializado em criar sites para restaurantes locais. Foco em resultados: mais pedidos, menos trabalho manual, presença online profissional.

## ✨ Características

- **Design Dark Premium** - Tema escuro (#0f1115) com destaque amarelo (#facc15)
- **100% Responsivo** - Mobile-first, funciona perfeitamente em qualquer dispositivo
- **WhatsApp Integrado** - Botão flutuante + CTAs estratégicos com mensagens pré-formatadas
- **Sites Demo Reais** - Links para projetos ao vivo (Lampião Burger, Restaurante Teste)
- **Copy Focada em Resultados** - Linguagem simples, sem jargão técnico
- **Seções Estratégicas:**
  - Hero com 3 variações de headline (A/B testing)
  - Demonstração com sites reais
  - Problemas que o cliente enfrenta
  - Soluções práticas
  - Garantias e confiança
  - Preços transparentes (Essencial R$80/mês, Profissional R$120/mês)
  - FAQ com 7 perguntas comuns
  - Sobre a equipe (Bryan + Dudu)
  - Como funciona (3 passos simples)

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Custom properties, Grid, Flexbox
- **JavaScript Vanilla** - Scroll effects, navegação mobile, animações
- **Google Fonts** - Inter (400-800)

## 🚀 Como Rodar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/SEU-USUARIO/portfolio-restaurantes.git
cd portfolio-restaurantes
```

2. Abra diretamente no navegador:
```bash
# Opção 1: Duplo clique no index.html
# Opção 2: Com servidor local (recomendado)
npx serve -l 3456
```

3. Acesse: `http://localhost:3456`

## 📦 Deploy na Vercel

1. Faça push pro GitHub
2. Entre em [vercel.com](https://vercel.com)
3. Clique em "Add New Project"
4. Selecione este repositório
5. Clique em "Deploy"

Pronto! Seu site estará no ar em segundos com HTTPS e CDN global.
